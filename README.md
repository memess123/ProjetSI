# Projet : Anthologie de 5 Mini-Jeux

Ce projet regroupe une collection de créations interactives. 

---

###  L'Équipe et les Créations

| Développeur | Jeu / Application | Description |
| :--- | :--- | :--- |
| **Ames ANDIAS** |  Esquive Infernale | Un jeu de survie où vous devez éviter des blocs rouges. |
| **Axel TANGUY** |  Simon Says | Testez votre mémoire avec ce grand classique. |
| **Cherif BOUBERRAGA** |  Calculatrice Fun | Une approche ludique du calcul basique. |
| **Ronan SEGALEN** |  Reflex Colors | Un défi de rapidité basé sur les changements de couleurs. |
| **Tom QUEMENEUR** |  Démineur | Le jeu de réflexion culte pour les amateurs de logique. |
